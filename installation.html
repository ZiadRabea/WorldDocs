How to install WorldLang?

Steps :

  1. Go to the language website at WorldLang and click the download icon
  2. Once the language is downloaded, click the downloaded file to install it,
  3. if you see a warning because the app is not on microsoft appstore, click more info and then run anyway
  4. copy the file path and paste it to the path variable
