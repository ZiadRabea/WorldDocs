Basic commands

Commands :

  1. translator :

    The "translator" command is designed to facilitate the translation of code snippets and programming language keywords between different programming languages.

  2. wcontroller :

    The "wcontroller" command empowers users to modify language keywords and customize them according to their preferences, thereby creating a personalized programming language experience. This feature grants users a sense of ownership and allows them to work with familiar terminology that aligns with their individual preferences.

  3. world :

    The "world" command enables you to log into the language and start writing code.
